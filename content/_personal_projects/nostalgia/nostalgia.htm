Nostalgia (2012)

call of duty world at war

 

Nostalgia is a non-profit multiplayer map pack for Call of Duty™: World at War, based on Medal of Honor™: Allied Assault.

The map pack consists of 4 multiplayer levels: Southern France, Remagen, Stalingrad and Algiers

 

My work on "Nostalgia" included:

  • Level Creation:
    • Recreating the base geometry of the original levels, using CoDWaWRadiant
    • Full prop- and set dressing
    • Texture creation using Adobe Photoshop and textures.com
    • Lighting pass
    • Collision pass
    • Sound and FX implementation

  • Scripting
    • Game mode implementation
    • AI Bot navigation implementation
