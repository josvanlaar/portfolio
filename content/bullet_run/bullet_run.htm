Bullet run (2012)

pC (FREE TO PLAY)
ACONY GmbH & Co KG | Level Designer

Bullet Run, previously known as Hedone, is a Free to Play multiplayer FPS utilizing the Unreal 3 engine. Set in the near future,
the biggest TV show on the planet offers everyone the chance to become famous, by creating the intense firefights and moves, basically everything an audience expects to see.

As a Level Designer on Bullet Run, I was in charge of designing and supporting several multiplayer levels:
"Carpark", "Drain", "Prison", "Vault", "Warehouse", "Wreck"

 

 

My work on Bullet Run included:

  • Researching timing, tactics and layout structures of competitive titles
  • Designing new multiplayer levels from scratch to finish
  • Updating existing layouts to a new theme, redesign where necessary
  • Creating and maintaining (level-) documentation
  • Level scripting and gameplay integration
  • Basic set / prop dressing
  • New asset integration

bullet run multiplayer maps

Carpark - Building an exciting layout based on a somewhat ordinary and real life location proved an interesting challenge. To facilitate air attacks, the roofs had to be opened as much as possible, resulting in an abandoned Carpark.

 

Drain - An old storm drain with many shanty type buildings forming the main structure of this level. As this was a reimagination of an existing layout, I had to find the balance between keeping what worked and what could use updates. The level is switching heavily between short- and long range attacks.

 

Prison - Inspired by Shutter Island, we wanted to create an abandoned prison on an island. This meant researching and implementing old-fashioned prison galleries and structures, providing effective sniper lanes for players.

 

Vault - This level was a reimagination from an existing layout,
and turned it into a Tibetan monastery. Heavily inspired byUncharted 2: Among Thieves, I had strike a good balance between the old layout and the new theme.

 

Warehouse - I provided support on this level, to upgrade its visuals and script the loading bay doors

 

Wreck - Another reimagination of an existing layout, but with a major twist; a huge freighter crashed through the map. This meant finding a fitting replacement for the existing routes, turning hallways, rooms and alleyways into a believable crash site.
